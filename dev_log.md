# AppSelecter v2 開発ログ (デスクトップ版)

## 記録日時: 2026-03-05 13:48

### 成功の境界線

- **Python + CustomTkinter による基盤構築**
  - ダークモード対応のモダンなUI設計（トースト風/設定画面）。
  - `config.py`, `settings.py` による定数・設定値の一元化（SSOT遵守）。
  - タイマー秒数を 5〜60秒 の範囲で可変に設定する機能。
- **拡張子連携とレジストリ操作**
  - **HKCU（ユーザースコープ）のみ**への最小限のレジストリ書き込みを実装。
  - レジストリ登録解除時に、変更前のデフォルトアプリ設定（バックアップ）を自動復元する機能を実装。
- **パッケージング**
  - PyInstaller `--onedir` モードにて `.exe` 化に成功。
  - 実行ファイルのディレクトリに `AppSelect_Settings.json` が生成されることを確認。

### 失敗の事象 (解決済み)

- CustomTkinter が初期インストールされていなかったため、`pip install` で追加。
- PyInstaller実行時、CustomTkinter のアセットファイル群（テーマ、フォントなど）が同梱されない問題に対し、`--add-data` オプションでモジュールパスを明示的に指定して解決。

### 自己チェックリスト (最終評価)

- [x] 新たなハードコーディングを追加していないか？ → 全て `config.py` に集約。
- [x] 既存のモジュールやCSSクラスを無視していないか？ → Pythonデスクトップアプリとして新規・最適化構築。
- [x] SSOTの原則に反していないか？ → はい。
- [x] コンソールエラー/警告はないか？ → `python main.py` および `.exe` 起動時に確認なし。

## 結論

Windows ローカル環境での「拡張子別のアプリセレクター」として、すべての要件を満たした完全なデスクトップアプリケーションが完成した。
